**Function 1: `selectChat(cdp, chatTitle)`**

Per D-03 and D-04, navigate Claude Code to a conversation by CDP-clicking inside #active-frame.

The CDP expression (store in `EXPRESSION` variable per conventions) must:
1. Access iframe: `const frame = document.getElementById('active-frame'); const doc = frame?.contentDocument || frame?.contentWindow?.document || document;`
2. Guard: `if (!doc) return { success: false, error: 'no_doc' };`
3. Find history/conversations toggle button inside `doc`:
   - Try: `doc.querySelector('button svg.lucide-clock, button svg.lucide-history, button svg.lucide-clock-rotate-left, button svg.lucide-list')?.closest('button')`
   - Fallback: search all visible buttons in `doc` where aria-label or title contains 'history' or 'conversations' or 'recent'
   - If found, click it and wait 600ms: `historyBtn.click(); await new Promise(r => setTimeout(r, 600));`
4. Search `doc` for elements whose `innerText.trim()` starts with `targetTitle.substring(0, Math.min(30, targetTitle.length))` — same heuristic as Antigravity version
5. Filter: skip hidden elements (`offsetParent === null`), skip containers with >5 children
6. Pick deepest matching element (max DOM depth), walk up to 5 parents to find cursor:pointer or button tag
7. Click the clickable element, return `{ success: true, method: 'clickable_parent' }` or `{ success: true, method: 'direct_click' }`
8. If no match found: `return { error: 'Chat not found: ' + targetTitle }`

Use the CDP context loop pattern (iterate `cdp.contexts`, try each context ID). Wrap expression in async IIFE with try/catch returning `{ error: e.toString() }`.

Use `JSON.stringify(chatTitle)` for safe interpolation into the expression string (same as Antigravity version).

**Function 2: `startNewChat(cdp)`**

Per D-05 and D-06, click the new chat button inside #active-frame.

The CDP expression must:
1. Access iframe same as above

   - Try icon: `doc.querySelector('button svg.lucide-plus, button svg.lucide-square-plus, button svg[class*="plus"]')?.closest('button')` — filter to visible, top area (rect.top < 200)

4. If not found: `return { error: 'New chat button not found' }`

Same context loop pattern. Async IIFE with try/catch.
