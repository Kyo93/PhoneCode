Replace the entire `checkChatStatus` function with:

```javascript
// --- Check Chat Status ---
async function checkChatStatus() {
    try {
        const res = await fetchWithAuth('/chat-status');
        const data = await res.json();

        chatIsOpen = data.hasChat || data.editorFound;

        if (!chatIsOpen) {
            // Auto-resume most recent chat for Claude Code (D-01: only when empty, D-02: newest by mtime)
            if (window.lastTarget === 'claude' && !autoResumeAttempted) {
                autoResumeAttempted = true;
                try {
                    const histRes = await fetchWithAuth('/chat-history');
                    const histData = await histRes.json();
                    if (histData.chats && histData.chats.length > 0) {
                        console.log('[CHAT] Auto-resuming most recent conversation:', histData.chats[0].title);
                        await selectChat(histData.chats[0].title);
                        return; // selectChat schedules its own loadSnapshot + checkChatStatus
                    }
                } catch (e) {
                    console.error('[CHAT] Auto-resume failed:', e);
                }
            }
            showEmptyState();
        } else {
            autoResumeAttempted = false; // Reset when a chat is open
        }
    } catch (e) {
        console.error('Chat status check failed:', e);
    }
}
```

**Why the guard matters:** `selectChat()` calls `setTimeout(checkChatStatus, 1000)` on success (line 995). Without `autoResumeAttempted`, this creates an infinite loop: checkChatStatus → selectChat → checkChatStatus → selectChat. The guard ensures auto-resume fires exactly once per empty-state detection. It resets when a chat is successfully opened.

**Per D-01:** Auto-resume only triggers when `!chatIsOpen`. If Claude Code already has a conversation open, `chatIsOpen` will be true and the auto-resume block is never entered.

**Per D-02:** `histData.chats[0]` is the most recent by mtime because `/chat-history` for Claude Code sorts `.jsonl` files by descending `mtime`.
